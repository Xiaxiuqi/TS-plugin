# 流水线提示词使用说明

本目录包含自动化生成前端状态栏项目的提示词。

## 🚀 极速上手（推荐）

使用 **`0_一键启动流水线.md`**：

1.  打开 AI 聊天窗口（Claude 3.5 Sonnet, GPT-4o 等能力较强的模型）。
2.  复制 **`0_一键启动流水线.md`** 的全部内容发送给 AI。
3.  紧接着发送你的 **角色卡内容**（可以直接复制 `角色卡.json` 的文本，或拖入文件）。
4.  AI 会自动按照“需求分析 -> 前端代码 -> 系统配置”的顺序，为你生成所有需要的文件代码。
5.  你只需要将 AI 生成的代码分别保存为 `前端.html`, `模板.json`, `提示词.txt` 即可。

---

## 🛠️ 分步定制（高级）

如果你需要更精细的控制，可以分步使用以下提示词：

1.  **阶段一**：发送 `1_需求分析提示词.md` + 角色卡 -> 获得《设计文档》。
2.  **阶段二**：发送 `2_前端定制提示词.md` + 《设计文档》 -> 获得 HTML 代码。
3.  **阶段三**：发送 `3_表格与系统提示词定制.md` + 《设计文档》 -> 获得 JSON 和 Prompt。
