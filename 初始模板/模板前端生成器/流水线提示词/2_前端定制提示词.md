## 3. 核心功能要求 (CRITICAL - 必须严格遵守)

### 3.1 数据源绑定 (Data Binding)
你**必须**严格模仿 `模板前端生成器/范例/前端美化.html` 中的数据处理逻辑。严禁使用 `window.dataSourceAPI` 或其他自定义方式，必须使用 `parent.AutoCardUpdaterAPI`。

**标准 boilerplate (直接复制并使用):**
```javascript
// 1. 获取 API 对象 (兼容 iframe 环境)
const dataSourceAPI = (typeof parent !== 'undefined' && parent.AutoCardUpdaterAPI) ? parent.AutoCardUpdaterAPI : {
    exportTableAsJson: function() { console.warn("No API found, using test data."); return null; },
    registerTableUpdateCallback: function(cb) { console.warn("No API found, no callbacks."); }
};

// 2. 数据处理函数 (将原始 JSON 转换为易用的对象)
function processJsonData(json) {
    if (!json) return null;
    const tables = {};
    for (const sheetId in json) {
        if (Object.prototype.hasOwnProperty.call(json, sheetId) && json[sheetId] && json[sheetId].name) {
            const sheet = json[sheetId];
            tables[sheet.name] = {
                headers: sheet.content[0] || [],
                rows: sheet.content.slice(1)
            };
        }
    }
    return tables;
}

// 3. 初始化与回调注册
window.onload = function() {
    // 初始加载
    const rawData = dataSourceAPI.exportTableAsJson();
    if (rawData) {
        const initialTables = processJsonData(rawData);
        renderUI(initialTables);
    } else {
        console.log("Waiting for data...");
    }

    // 注册自动更新
    dataSourceAPI.registerTableUpdateCallback(function(newData) {
        console.log("Detected data update, refreshing UI...");
        const updatedTables = processJsonData(newData);
        renderUI(updatedTables);
    });
};
```

### 3.2 交互反馈 (Interaction)
**核心要求**：界面中的关键元素必须是**可点击**的，并触发消息发送。
1.  **点击即发送**：用户点击“地点”、“NPC”、“物品”或“选项按钮”时，必须调用 `sendGameActionRequest` 发送对应的文本指令。
2.  **指令格式**：指令应清晰表达意图，例如 `前往 [地点名]`、`与 [NPC名] 交谈`、`使用 [物品名]`。
3.  你**必须**使用以下标准函数来发送指令，严禁直接操作 DOM 输入框。

**标准 boilerplate (直接复制并使用):**
```javascript
function sendGameActionRequest(msg) {
    console.log("Action sent:", msg);
    // 尝试调用父级 API
    if (typeof parent.triggerSlash === 'function') {
        try { parent.triggerSlash('/setinput ' + msg); } 
        catch(e) { console.error(e); }
    } else {
        // 回退：尝试寻找输入框
        const input = parent.document.getElementById('send_textarea');
        if (input) { input.value = msg; } 
        else { alert(`[模拟发送] ${msg}`); }
    }
}
```

### 3.3 渲染逻辑 (`renderUI`)
*   `renderUI(tables)` 函数接收的 `tables` 对象，其键为 **表名** (如 "官员档案", "全局数据表")。
*   访问方式必须是 `tables["表名"]`。
*   每个表包含:
    *   `.headers`: 字符串数组 (列名)。
    *   `.rows`: 二维数组 (数据行)。
*   **禁止** 使用 `json.tables` 或 `headers.indexOf` 硬编码索引 (除非你先动态获取了 headers)。建议使用 helper 函数：
    ```javascript
    const getVal = (row, headers, colName) => row[headers.indexOf(colName)];
    ```
