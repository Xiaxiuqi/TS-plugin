# Macro - 宏大世界模板

## 适用范围

完整国家、星球、多文明世界，如：
- 国家、王国、帝国
- 独特文明星球
- 多星系星际联盟
- 神话奇幻大陆
- 多元宇宙

## 核心条目结构

### uid: 001 - 元数据

```yaml
uid: 001
comment: 世界元数据
content: |
  world_name: [世界名称]
  world_type: [类型：国家/星球/星系/多元宇宙]
  theme_tags: [主题1] | [主题2] | [主题3]
  tone: [基调1] | [基调2]
  era: [时代背景]
  tech_level: [科技/魔法水平]
position: before_char
enabled: true
depth: 4
```

## 条目数量建议

- 核心条目：5-10个
- 区域条目：10-20个
- 文明/国家条目：5-15个
- 种族条目：3-10个
- 势力条目：5-15个
- 地点条目：10-30个
- 人物条目：10-30个

**总计**：40-100+个条目
