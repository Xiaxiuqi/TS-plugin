# 世界观规模判断提示词

## 阶段目标

根据用户的创作需求，判断世界观的规模类型，并引导用户确定核心框架。

## ⚠️ 重要规则：确认后再推进

**每个阶段结束前，必须获得用户明确确认！**

- 确认后才记录条目清单到Plan文件
- 确认后才进入条目生成阶段
- 用户沉默或模糊回答时，继续追问或等待

## 三种规模类型

### Micro - 微观场景

**定义**：单一场所或小型封闭环境

**适用场景**：
- 废弃医院、鬼屋、 haunted house
- 特定型号飞船、潜艇
- 神秘商店、特殊设施
- 封闭研究基地
- 异空间碎片

**特点**：
- 结构简单，通常线性或小型迷宫
- 注重氛围营造和细节描写
- 条目数量：5-15个
- 核心条目：规则、环境、机制

**必需要素**：
- 核心规则（进入/离开条件、行为规范）
- 环境描述（布局、感官体验）
- 特殊机制（如果有）
- 主要危险源

### Mid - 中型区域

**定义**：城市及周边、组织势力范围

**适用场景**：
- 城市、城镇
- 商业帝国、公司
- 地下帮派势力范围
- 魔法森林、特殊区域
- 学院、军事基地

**特点**：
- 多个地点，有势力关系
- 存在社会结构
- 条目数量：15-40个
- 核心条目：规则、环境、势力、地点

**必需要素**：
- 核心规则（法律、社会规范）
- 区域划分（各区域概览）
- 势力/组织（主要势力）
- 重要地点（地标建筑）

### Macro - 宏大世界

**定义**：完整国家、星球、多文明世界

**适用场景**：
- 国家、王国
- 独特文明星球
- 多星系星际联盟
- 神话奇幻大陆
- 多元宇宙

**特点**：
- 复杂政治，多文明
- 宏大历史背景
- 条目数量：40-100+
- 核心条目：规则、世界生态、文明、历史、势力

**必需要素**：
- 核心规则（世界法则、物理/魔法规则）
- 世界生态（地理、气候、资源）
- 文明/势力（主要文明体）
- 历史脉络（重大事件）
- 重要区域（大陆/星球概览）

## 判断引导对话

### 规模判断提问

```
根据您的描述，我需要确定世界观的规模。请选择最接近您想法的选项：

1. **微观场景** - 故事主要发生在一个特定的地点
   例如：一座废弃医院、一艘飞船、一个神秘商店

2. **中型区域** - 故事涉及一个城市或组织的范围
   例如：一座城市、一个商业帝国、一个魔法森林

3. **宏大世界** - 故事涉及国家、星球或更大的范围
   例如：一个王国、一个星球文明、星际联盟

您倾向哪种规模？如果不确定，可以描述您希望故事发生的范围，我来帮您判断。
```

### 根据用户回答确认

**Micro确认**：
```
您选择了【微观场景】。

这意味着我们将专注于一个特定地点的详细设定。请告诉我：

1. 这是什么类型的场所？（建筑/设施/自然区域/其他）
2. 大致的氛围是怎样的？（恐怖/神秘/科幻/日常...）
```

**Mid确认**：
```
您选择了【中型区域】。

这意味着我们将构建一个城市或组织级别的世界。请告诉我：

1. 这是什么类型的区域？（城市/组织领地/特殊区域）
2. 有没有初步的势力或组织设想？
```

**Macro确认**：
```
您选择了【宏大世界】。

这意味着我们将构建一个宏大的世界观。请告诉我：

1. 世界的大致规模？（国家/星球/星系/多元宇宙）
2. 主要的文明或势力类型？
```

## 阶段结束确认

```
让我确认一下框架：

**规模类型**：[Micro/Mid/Macro]
**核心条目数量**：[X]个
**次要条目数量**：约[Y]个

**核心条目清单**：
- 001: [标题]
- 002: [标题]
- ...

这个框架是否符合您的需求？确认后我们将开始逐条生成内容。
```

## 下一步行动

框架确认后：

1. 读取对应的模板文件（micro/mid/macro-world-template.md）
2. 读取 `3-世界观框架.md`
3. 开始生成核心条目
