# prompts 索引

`prompts/` 是奶龙王一键制卡的流程提示词目录。

| 文件                 | 作用                                    |
| ------------------ | ------------------------------------- |
| `0-总览与启动.md`       | 从零写卡入口                                |
| `1-需求分析.md`        | 明确需求、题材、边界与组件范围                       |
| `2-世界观规模判断.md`     | 判断 Micro/Mid/Macro 等世界观规模             |
| `3-世界观框架.md`       | 搭建世界观结构                               |
| `4-人物设计.md`        | 设计主要角色与人物关系                           |
| `5-剧情设计.md`        | 设计剧情阶段、事件链与玩法循环                       |
| `6-表格模板讨论.md`      | 设计表格模板                                |
| `7-开场白状态栏讨论.md`    | 组件规划总控                                |
| `7-1-开场白前端.md`     | 开场白前端制作                               |
| `7-2-状态栏前端.md`     | 状态栏前端制作                               |
| `7-3-剧情推进预设.md`    | 剧情推进预设制作                              |
| `7-4-剧情推进前端.md`    | 剧情推进前端制作                              |
| `7-5-头像与立绘前端增强.md` | 头像、立绘、CG 展示增强                         |
| `8-打包确认.md`        | 打包前确认与交付                              |
| `9-修改卡启动.md`       | 修改已有角色卡入口                             |
| `10-MVU迁移到数据库.md`  | MVU 卡迁移为 shujuku 数据库卡的子流程（骨架版 + 边做边补） |

执行原则：先由 `README.md` 分流，再由入口 prompt 接管，具体阶段按前置阅读要求执行。
