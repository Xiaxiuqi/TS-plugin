这是项目级通用的表格可读视图入口。每张表会拆成独立 Markdown，方便直接阅读。

## 使用说明

- `readable/*.md` 不是纯预览文件，而是可编辑工作文件。
- 允许修改的部分：`## 说明`、`## 更新规则`、`## 当前内容`、`## 更新配置`、`## 导出配置`。
- `## 更新规则` 必须保持四条固定键名：`初始化 / 更新 / 插入 / 删除`。
- `## 更新配置` 必须保持合法 JSON 代码块格式。
- `## 导出配置` 必须保持合法 JSON 代码块格式。
- 暂不允许修改的部分：表名、uid、orderNo。
- `00-目录.md` 中的 `## 全局注入配置` 也可编辑，会回写到 `mate.globalInjectionConfig`。
- 回包时会自动执行：`readable/*.md + 00-目录.md -> embedded-template.pretty.json -> 注入角色卡`。
