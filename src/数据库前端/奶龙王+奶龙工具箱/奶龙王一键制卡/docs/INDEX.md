| `人设优化/` | 人物设定优化资料 |
| `瑟瑟语料/` | 特定风格语料资料 |
| `nsfw状态栏/` | 状态栏相关资料 |
| `图片与前端知识库/` | 图片、气泡、头像、立绘、前端调用知识 |
