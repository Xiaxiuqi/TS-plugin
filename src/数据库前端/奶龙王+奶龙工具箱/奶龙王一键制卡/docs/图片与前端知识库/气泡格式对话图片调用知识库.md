# 前端接口知识库

## BubbleCG — CG 图片接口

通过 `window.top.BubbleCG` 调用，在正则脚本的 iframe 中使用。

### 获取指定 CG

```
const url = await window.top.BubbleCG.getImage('组名', 序号);
```

示例：`await window.top.BubbleCG.getImage('天之音', 1)` 获取天之音组第1张。

### 获取随机 CG

```
const url = await window.top.BubbleCG.getRandomImage('组名');
```

示例：`await window.top.BubbleCG.getRandomImage('天之音')` 随机获取天之音组一张。

## BubbleAvatar — 角色头像接口

通过 `window.top.BubbleAvatar` 调用。

### 获取角色主头像

```
const url = await window.top.BubbleAvatar.getAvatar('角色名');
```

自动回退到全局头像。

### 获取情绪差分头像

```
const url = await window.top.BubbleAvatar.getMoodAvatar('角色名', '情绪');
```

支持的情绪：喜悦、愤怒、悲伤、紧张、平和、害羞、嫌弃、爱恋。

### 获取角色主题色

```
const color = await window.top.BubbleAvatar.getColor('角色名');
// 返回 '#58a6ff' 或 null
```

## 正则脚本示例

### 头像获取 `{{avatar:角色名|情绪}}`

- 正则：`/\{\{avatar:(\S+?)(?:\|(\S+?))?\}\}/gi`
- 捕获组：$1=角色名，$2=情绪（可选）
- 用法：`{{avatar:望月雫}}` 或 `{{avatar:望月雫|喜悦}}`

### CG 随机图片 `{{cg-random:组名}}`

- 正则：`/\{\{cg-random:(\S+?)\}\}/gi`
- 捕获组：$1=组名
- 用法：`{{cg-random:天之音}}`

### CG 指定图片 `{{cg:组名#序号}}`

- 正则：`/\{\{cg:(\S+?)#(\d+)\}\}/gi`
- 捕获组：$1=组名，$2=序号
- 用法：`{{cg:天之音#1}}`

### 注意事项

- 所有 API 都需要通过 `window.top` 访问
- 在 iframe 中需轮询等待 API 就绪：`if(!window.top.BubbleCG){ setTimeout(run, 500); return; }`
