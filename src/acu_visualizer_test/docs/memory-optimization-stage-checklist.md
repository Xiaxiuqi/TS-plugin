# ACU Visualizer 内存优化阶段验收表

本表用于每个内存优化阶段完成后确认功能、CSS、数据兼容和回滚条件。任何阶段未通过时，不进入下一阶段。

## 1. 通用验收原则

- 不修改正式版 `public/acu_visualizer/acu_visualizer.js`。
- 不修改正式版 `public/acu_visualizer/acu_visualizer-test.js`。
- 不改变测试版推荐加载入口。
- 不改变数据库 API 语义。
- 不直接删除用户快照、历史记录等持久数据。
- 不改变现有 DOM class 和 CSS selector。
- 每阶段必须可单独回滚。

## 7. 暂停条件

出现以下任意情况，立即停止当前阶段并回滚：

- CSS 视觉异常。
- 表格无法加载。
- 删除行或保存数据库异常。
- 搜索结果错误。
- 行排序错误。
- localStorage 旧数据无法读取。
- 历史记录丢失。
- 自动加载在 iframe 环境下失效。
